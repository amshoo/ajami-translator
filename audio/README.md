# Audio Files

This folder contains community-recorded Hausa pronunciation audio for the Ajami Translator dictionary.

## Naming Convention

Every file is named after the Hausa word it contains, with spaces replaced by underscores:

```
gida.mp3          →  "house"
ruwa.mp3          →  "water"
na_gode.mp3       →  "thank you"
mai_kyau.mp3      →  "good"
```

The filename must match exactly what is in the `audio` field of `dictionary.json`. Since all entries are pre-filled with the expected filename, **you only need to upload the audio file — no JSON editing required**.

## How to Contribute

1. Open the [Audio Contribution page](../contribute.html) in your browser
2. Pick a word, record it, and download the file
3. Upload the file here via a Pull Request

## Format

- Format: MP3 preferred, WebM or OGG also accepted
- Duration: 1–3 seconds per word
- Quality: Clear, natural speech in a quiet environment
- Speaker: Native Hausa speakers preferred, all contributions welcome
