What is Ajami?

Ajami is the practice of writing African languages using Arabic script. Hausa speakers have used it since the 17th century for poetry, trade, and Islamic scholarship.

⚙️

How it works

English is first translated to Hausa using Meta's NLLB-200 model. The Hausa text is then converted to Ajami script using phoneme-to-glyph mapping from the West African Warsh tradition.

⚠️

Accuracy note

Machine translation to Hausa is imperfect. Ajami rendering is rule-based and may miss regional variants. Use as a learning aid, not a certified translation.
